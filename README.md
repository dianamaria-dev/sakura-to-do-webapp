# Sakura To-Do App 🌸

A soft sakura-themed to-do app built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Move tasks between categories
- Delete tasks
- Save tasks with localStorage
- Random motivational quote on refresh
- Sakura-inspired background and falling petals
- Desktop-first layout

## Sections
- New Tasks
- Priority Task
- Important Task
- Small Task

## Tech Used
- HTML
- CSS
- JavaScript

## Notes
This project was designed primarily for laptop/desktop view.
